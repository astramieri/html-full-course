## Lessons

1. Introduction
2. Head Element
3. Text Basics
4. List Types
5. Links

## Main Topics

- ```<html>``` (lang)
- ```<head>```
	- ```<meta>``` (charset, author, description, viewport)
	- ```<link>``` (icon, stylesheet)
	- ```<title>```
- ```<body>```
	- block level (start on new line with space around it)
		- ```<h1>```, ```h2>```, etc. (headings)
		- ```<hr>``` (horizontal rule)
		- ```<br>``` (single line break)
		- ```<ol>``` (ordered list)
		- ```<ul>``` (unordered list)
		- ```<dl>``` (description list)
	- inline level
		- ```<em>``` (emphasize)
		- ```<strong>```
		- ```<abbr>``` (abbreviation)
		- ```<a>``` (anchor)
	- semantic element
		- ```<section>```
		- ```<nav>``` (navigation)
	- entities
		- used to display reserved characters
		- begins with an ampersand (&) and ends with a semicolon (;)
		- examples:
			- ```&nbsp;``` (space)
- white space collapse
